7.1 gitbook配置使用

7.1 gitbook配置使用

1 安装

  • 环境:
    Ubuntu 16.04.5 LTS
  • 安装:
    sudo apt-get install npm
    sudo npm install -g gitbook-cli
    sudo apt-get install node
    sudo ln -s /usr/bin/nodejs /usr/bin/node
  • 验证:
    gitbook -V

    2 配置

  • 创建仓库:
    mkdir mybook
  • 编辑目录文件:
    vim SUMMARY.md
